Tablas

Definicion

Las tablas en HTML sirven para mostrar contenido estructurado.

Semantica

La semantica basica de una tabla se compone de:

  • table: Etiqueta que encierra una tabla
  • tr: Table Row. Etiqueta que construye una fila.
  • td: Table Data. Etiqueta que construye una celda.

Construccion de una Tabla

Estrucutura base:

Para construir una tabla, se da el primer paso con la etiqueta table, dentro de la misma, se iniciará la etiqueta tr, la cual indica el inicio de una fila. Para dar contenido a las celdas de la misma, se usara la etiqueta td por cada celda.

Etiquetas adicionales

  • Titulo: Se usan mediante la etiqueta caption, y se colocan justo despues de la etiqueta table.
  • Cabeceras: Se establecen con la etiqueta thead y en vez de usar las td, usaremos la etiqueta th por cada celda a definir.
    • Cuerpo de la tabla: Al definir una cabecera, damos por hecho que se debe distinguir cual el es cuerpo de la tabla. Para esto, se usa la etiqueta tbody.
  • Pie de Tabla: Establece un pie o finalizacion para la tabla. Se usa la etiqueta tfoot.

Atributos

Para que un elemento ocupe mas de una fila o mas de una columna, tenemos disponibles 2 atributos:

  • colspan: Permite que la celda ocupe mas columnas que la asignada. El valor por defecto es 1
  • rowspan: Permite que la celda ocupe mas filas que la asignada. Cuando modificamos el valor, las celdas ocuparan espacio moviendose hacia abajo, por eso es necesario que se elimine la celda que sobrará para evitar que la tabla se "rompa". El valor por defecto es 1
