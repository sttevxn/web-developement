¿Que son las listas?

Las listas son herramientas que nos ofrece HTML para listar elementos

.En funcion de necesidades e importancia, existe 3 tipos de listas.

Tipos de listas:

1- Unordered list

Son utilizadas cuando el orden de los elementos no ifluye por ejemplo, una lista de compras

Semantica:

Para hacer uso de las UL, se usan las etiquetas "ul", y dentro de la misma se enlistaran los elenentos de se desean haciendo uso, para cada ítem, de la etiqueta "li".

2-Ordered list

Contrariamente al caso anterior, el orden de los elementos influyen y tienen importancia, por ejemplo un top 10 global

Semantica:

Para hacer uso de las UL, se usan las etiquetas "ol", y dentro de la misma se enlistaran los elenentos de se desean haciendo uso, para cada ítem, de la etiqueta "li".

3- Definition List

Son utilizadas para hacer una lista de definiciones, por ejemplo, un diccionario.

Semantica:

La semantica de las listas de definicion se compone por dos etiquetas: "dt" (definition term) y "dd" (definition descriptionj)

Listas Anidadas:

Las listas vistas anteriormente tienen la capacidad de ser anidadas entre si segun se necesite dicho comportamiento. Dando una imagen jerarquica de los contenidos enlistados.
